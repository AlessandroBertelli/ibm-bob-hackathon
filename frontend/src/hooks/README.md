# Hooks

This directory contains custom React hooks for shared logic.

## Hooks to Implement

- **useAuth**: Authentication state and methods
- **useGroup**: Group management operations
- **useSwipe**: Swipe gesture handling
- **useRestaurants**: Fetch and manage restaurant data
- **useWebSocket**: Real-time updates for group sessions
- **useLocalStorage**: Persistent local storage management

## Naming Convention

- Use camelCase with 'use' prefix: `useAuth.ts`, `useSwipe.ts`
- Export the hook as a named export
- Include TypeScript types for hook parameters and return values
