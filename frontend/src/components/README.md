# Components

This directory contains reusable React components.

## Structure

- **Common Components**: Buttons, inputs, cards, modals, etc.
- **Feature Components**: Restaurant cards, swipe interface, group management, etc.
- **Layout Components**: Header, footer, navigation, etc.

## Naming Convention

- Use PascalCase for component files: `RestaurantCard.tsx`
- Use descriptive names that indicate the component's purpose
- Group related components in subdirectories when appropriate
